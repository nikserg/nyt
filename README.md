# NYT

## Requirements

- Laravel Herd

OR

- PHP 8.2
- Composer

## Installation 

Project could be run with Laravel Herd (https://herd.laravel.com/)
or by running
```bash
composer install
php artisan serve
```
if you have local PHP.

## Usage

Use Postman or similar tool to send GET request to 

`http://<your host>/api/v1/bestsellers`
